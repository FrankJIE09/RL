# DH参数校准脚本使用说明

## 安装依赖

```bash
pip install roboticstoolbox-python spatialmath-python numpy matplotlib
```

## 使用方法

```bash
python3 dh_calibration.py
```

## 脚本功能

1. **创建机器人模型**：使用Robotics Toolbox创建2关节机器人
2. **设置参数**：
   - 真实DH参数（用于生成"测量"数据）
   - 初始估计参数（带误差，用于校准）
3. **生成测试数据**：模拟3D相机测量末端位置
4. **执行校准**：使用Gauss-Newton方法迭代优化参数
5. **评估结果**：比较真实参数和估计参数

## 主要公式实现

- **正向运动学**：`-1T_{n+1} = -1T_0 × 0T_1 × ... × nT_{n+1}`
- **雅可比矩阵**：基于书中公式14.14-14.18（螺旋参数方法）
- **最小二乘**：`Δφ = (J^T × J)^{-1} × J^T × Δy`
- **迭代更新**：`φ^{k+1} = φ^k + Δφ`

## 输出

- 校准过程的迭代信息
- 参数误差对比
- 位置误差统计
- 误差收敛曲线图（保存为PNG）
